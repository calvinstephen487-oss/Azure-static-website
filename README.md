# Azure-static-website

### What is Azure?
Azure is a cloud computing platform developed by Microsoft that provides a wide range of services including computing, storage, networking, and analytics. It enables individuals and organizations to build, deploy, and manage applications through Microsoft's global network of data centers on a pay-as-you-go model.

### What is an Azure Storage Account?
An Azure Storage Account is a resource that allows you to store all kinds of data objects including blobs, files, queues, and tables. It acts as a container for managing these storage services as a group, offering features such as durability, scalability, high availability, and various configuration options like performance tiers, redundancy, and access controls.

### What is Azure Static Website Hosting?
Azure Static Website Hosting is a feature of Azure Storage that enables you to host static web content—such as HTML, CSS, and JavaScript files—directly from a storage account. With this feature, your static files are served globally over HTTP/HTTPS, allowing for fast, cost-effective, and scalable website hosting without the need for managing traditional web servers.
